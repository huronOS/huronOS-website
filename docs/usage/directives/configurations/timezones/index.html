Timezones

The directives file is able to configure a timezone for the huronOS instance via the TimeZone directive, this timezones are compliant with the IANA timezones standard, we do highly recommend you checking which timezone is your city.

huronOS Queue 0.3 is compliant with the 2023c release of the timezones database.

Example​

TimeZone=America/Mexico_City
